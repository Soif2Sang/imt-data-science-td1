# Donnees

Les donnees brutes ne sont pas versionnees dans Git car elles sont volumineuses.

Pour utiliser le projet, placer les dossiers suivants a la racine du repo :

```text
Events/
Plantar_activity_reference/
```

Structure attendue :

```text
Events/
└── S01/
    └── Sequence_01/
        └── classif.csv

Plantar_activity_reference/
└── S01/
    └── Sequence_01/
        └── insoles.csv
```

Le dossier `Plantar_activity/` n'est pas necessaire pour ce repo.

Si vous partagez les donnees entre camarades, utilisez un lien externe ou une archive separee plutot qu'un commit Git.
